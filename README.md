# README

## 简介

该项目是一个基于TMS320的BSP（Board Support Package），旨在提供一个通用的硬件抽象层，使得开发者可以更加方便地开发TMS应用。

## 文件结构

- `README.md`: 本文件，提供项目说明
- `Inc/`: BSP头文件
- `Src/`: BSP源代码

## 使用方法

1. 克隆或下载该项目源代码。

    ```bash
    git clone https://github.com/wuyi03/lib_dsp.git
    ```

2. 在你的项目中添加该BSP的源代码。

3. 在你的应用程序中使用BSP提供的API进行开发。
